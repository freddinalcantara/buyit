## Technologies

  - HTML5 , CCS3, Bootstrap 3.71
  - Frameworks: Angular Js

## Instalation

1. Clone the project

	``` git clone git@github.com:freddinalcantara/buyit.git ```

2. Install all the dependencies

	``` npm install ```

3. Start the project

	```npm start```

4. Open the project 

	[http://localhost:8080/](http://localhost:8080/)
